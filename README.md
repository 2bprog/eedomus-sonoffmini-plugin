Plugin pour utiliser SonOff Mini en mode DIY, avec retour d'etat, auto-off, 

Version Original de Tibalt76, Modifiée par Benjamin B. (Benj70b)

## Prérequis

* Intégrer le module sur le WIFI local voir tuto de ![Planete-domotique]( https://www.planete-domotique.com/blog/2019/10/28/sonoff-mini-diy-eedomus/)
* Connaitre l'IP et le Device ID de votre SonOff Mini

## Champs à configurer : 

* **IP** : IP adresse de votre SonOff Mini
* **Device ID** : Identifiant de votre SonOff Mini
